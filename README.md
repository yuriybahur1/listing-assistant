# Listing Assistant

Private internal tool for authorized shop owners.

## Purpose

Listing Assistant is intended for the project team and shop owners who explicitly authorize access through Etsy OAuth.

The application uses Etsy Open API v3 to retrieve shop and listing data from authorized shops and to create and update product listings.

It helps prepare listing drafts and listing-content suggestions for authorized shop owners.

## Data Use

The application accesses only the data required to provide its intended functions.

It does not resell Etsy data or send marketing emails.

## Etsy Relationship

Listing Assistant is not affiliated with, endorsed by, or sponsored by Etsy.
